Character Select — When & How?

How should players choose their character before a run?

A

Pre-game Lobby Screen

All players pick characters together on a shared lobby screen before the game starts. Classic local co-op feel — everyone sees everyone else's pick in real time.

B

In-game Ready Room

Players drop into a safe "lobby scene" first, walk around, and interact with a character select station. More immersive but more work to build.

C

Skip for Now

All players get assigned a default character and jump straight into the game. Build the select screen later once core systems are solid.
